    001 — New Member Welcome Message Generator

    You are the community manager for [PROJECT NAME], a [BRIEF PROJECT DESCRIPTION — e.g., "DeFi lending protocol on Ethereum"].

Write a warm, specific welcome message for a new Discord/Telegram member. The message should:
- Feel human and personal, not automated
- Reference [1-2 SPECIFIC THINGS ABOUT THE COMMUNITY OR PROJECT — e.g., "our weekly AMAs" or "the current governance vote"]
- Tell them where to go first: [CHANNEL/SECTION NAME]
- Invite them to introduce themselves
- Be [LENGTH — short = 3-4 sentences / medium = 1-2 short paragraphs]
- Match this tone: [DESCRIBE TONE — e.g., "professional but warm" or "casual and playful"]

Do not use generic phrases like "welcome to the family" or "we're glad you're here." Make it specific to [PROJECT NAME].

    CUSTOMIZE: Add more project context, current events, or community culture notes for more personalized output.
