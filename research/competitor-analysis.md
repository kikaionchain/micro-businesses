## 3. Naming Conventions (Patterns + Examples)

### Pattern 1: "[X] OS" (Operating System Metaphor)
Signals: comprehensive, systematic, authoritative
- **Crypto OS** - "Your All-In-One Crypto Operating System"
- Life OS (Notion - top seller)
- Creator OS (common in creator economy)
- Second Brain (Tiago Forte concept, widely copied)

### Pattern 2: "The [Noun] Suite"
Signals: professional toolkit, bundled value
- **The Trading Suite** (fxpabs)
- **The Trading Suite Pro** (upgrade path)
- The Core Bundle (visual AI prompts)
- Note: "Suite" = software world vocabulary, implies professional grade

### Pattern 3: "Ultimate [Descriptive] [Product Type]"
Signals: comprehensive, definitive, beats competition
- **Ultimate Trading Journal**
- Ultimate Crypto Guide
- The Ultimate ChatGPT Prompt Bundle
- Caution: overused, losing signal strength

### Pattern 4: Descriptor + Compass/Hub/Vault/Tracker
Signals: organized, directional, focused tool
- **Finance Compass 2.0** (direction metaphor = smart)
- Crypto Trading Hub
- Alpha Vault (crypto community product)
- AI Mastery Vault (prompt library)
- The Prompt Performance Pack

### Pattern 5: "[Adjective] + [Niche] + [Product Type]"
Clean, SEO-friendly, descriptive
- **Crypto Trading Journal Lite**
- Advanced Crypto Futures Trading Journal
- Professional Options Trading Journal
- Beginner Cryptocurrency Technical Analysis Course

### Pattern 6: Power Word Suffixes (High-Converting Vocabulary)
- **Vault** - implies exclusive, locked, precious
- **Bible** - definitive reference, comprehensive
- **Blueprint** - actionable, step-by-step
- **Protocol** - systematic, follows rules
- **Playbook** - strategies, proven moves
- **Arsenal** - weapons/tools for battle
- **System** - replicable, engineered
- **Code** - secret, insider, unlock
- **Compass** - direction, guidance

### Pattern 7: Creator Personal Brand Names
High-performer examples with unique single names:
- **Easlo** (Notion templates, $500K+ revenue) - made-up, memorable
- **Traf** (iOS icon set, $1M week) - short, distinctive
- **Rekt Capital** (crypto analysis) - crypto native terminology
- **Daniel Vassallo** - real name + "The Small Bets" branding

### Product Name Analysis: What Works in Crypto Specifically
Top-performing crypto-native vocabulary:
- "Alpha" (insider knowledge)
- "Degen" (community signal, self-aware)
- "On-chain" (technical legitimacy)
- "DeFi" (specific ecosystem)
- "HODL / Long / Short" (trading culture)
- "Signal" (valuable insight)
- "Protocol" (systematic approach)

**Winning formula for crypto products:** `[Crypto Power Word] + [Function] + [Product Type]`
Examples: *DeFi Protocol Playbook*, *Alpha Signal OS*, *On-Chain Research Vault*

## 4. Design Inspiration (Colors, Styles, Approaches)

### 2024-2026 Crypto Landing Page Trends

#### Color Palettes That Dominate
1. **Dark Mode + Neon Accent** (dominant pattern)
   - Background: #0A0A0A or #0D0D14 (near-black with blue tint)
   - Accent: Electric blue (#00D4FF), neon green (#00FF7F), or gold (#FFD700)
   - Text: White primary, gray secondary
   
2. **Deep Navy + Purple** (premium/DeFi aesthetic)
   - Background: #0F0F23 or #1A1A3E
   - Accent: Purple/violet (#8B5CF6), cyan (#06B6D4)
   
3. **Black + Gold** (luxury/serious trading)
   - Background: Pure black
   - Accent: Gold/amber (#F59E0B)
   - Signals: premium, institutional

4. **Clean White + Minimal** (Notion/productivity cross-over)
   - Used by Easlo and productivity-focused templates
   - Less "degen", more mainstream appeal
   - Pastel accents possible

#### Typography Patterns
- **Large, bold hero headlines** - 64-96px, condensed fonts
- **Gradient text** on key words (e.g. headline word in gradient)
- **Monospace/code fonts** for technical credibility (JetBrains Mono, Fira Code)
- **High-contrast** text always (white on dark)
- Fonts spotted: Inter, Space Grotesk, Clash Display, Syne

#### Layout Patterns
- **Hero section:** One killer headline + subheadline + CTA (no clutter)

## 5. Key Takeaways

### What We Should Do (Top 5 Strategic Insights)

**1. Price in the $25-$49 sweet spot for Notion templates**
- Not $9 (signals low value, buyer remorse for sellers)
- Not $99 (too much friction for unestablished brand)
- $29-$49 = high perceived value, low enough for impulse
- Add a FREE "lite" version to build email list and trust

**2. Name products using crypto-native power words + function**
- Formula: `[Power Word] [Function] [Product Type]`
- Examples: "Alpha Journal OS", "DeFi Research Vault", "Signal Protocol Kit"
- Avoid generic names like "Crypto Notion Template" - drowns in search
- Use "OS" for comprehensive systems, "Vault" for collections, "Playbook" for guides

**3. Dark mode is non-negotiable for crypto audience**
- Near-black backgrounds with 1-2 neon accent colors
- Every page should feel like it belongs in the ecosystem
- Consider 8 distinct color palettes (one per micro-business) to differentiate
- Typography: Bold condensed headlines, monospace for code/technical elements

**4. Stack value and bundle aggressively**
- God of Prompt went from "$9 prompt pack" to "$150 complete bundle" - this is the move
- Create a "base" product ($25-39) + "pro" version ($49-69) + bundle ($79-99)
- Include video walkthroughs inside the product to justify premium price
- "What's inside" lists should be exhaustive - the longer the better

**5. Build Twitter/X following BEFORE launch**
- Every top Gumroad seller (Easlo, Traf, Daniel Vassallo, Rekt Capital) has large X following first
- Content showing the product in use (before/after) = best marketing
- Crypto audience is ALREADY on X - meet them there
- Launch with 24-hour pricing urgency + ask buyers to post screenshots
